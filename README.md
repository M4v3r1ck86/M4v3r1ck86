## Engineering principles

I like to build data solutions that are:

- **Reproducible:** one-command local execution whenever possible
- **Observable:** logs, metrics, validations and failure visibility
- **Scalable:** designed for increasing data volume and processing complexity
- **Governed:** documented lineage, ownership, quality rules and data contracts
- **Cost-aware:** cloud resources and architecture choices aligned with business value
- **Production-oriented:** tests, CI, Docker, clear documentation and maintainable code
